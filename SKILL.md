# Amazon PPC Bid Optimizer

## Core Strategy
1. Structure: Auto + Manual Exact + Manual Phrase + Manual Broad
2. Harvest search terms from auto-campaign weekly
3. Move high-converting terms to exact match
4. Set negative keywords for wasted spend
5. Target ACOS < 25%

## Bid Management
- Start at suggested bid
- Adjust +20% daily based on performance
- Top-of-search: +50% for converting terms
